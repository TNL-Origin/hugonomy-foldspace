# VibeAI FoldSpace Installation Guide

## Requirements
- Google Chrome (version 88 or later)
- Windows, Mac, or Linux

## Installation Steps

### Step 1: Extract the Package
If you downloaded a ZIP file, extract it to a folder on your computer.

### Step 2: Open Chrome Extensions
1. Open Google Chrome
2. Type `chrome://extensions/` in the address bar
3. Press Enter

### Step 3: Enable Developer Mode
1. Look for the "Developer mode" toggle in the top-right corner
2. Click it to turn it ON (it should turn blue)

### Step 4: Load the Extension
1. Click the "Load unpacked" button (appears after enabling Developer mode)
2. Navigate to the extracted folder
3. Select the `dist` folder (NOT the parent folder)
4. Click "Select Folder"

### Step 5: Verify Installation
You should see "VibeAI FoldSpace" appear in your extensions list with version 2.14.6.

### Step 6: Pin the Extension (Optional)
1. Click the puzzle piece icon in Chrome's toolbar
2. Find "VibeAI FoldSpace"
3. Click the pin icon to keep it visible

## First Use

### Consent Modal
1. Visit any supported platform (ChatGPT, Claude, Gemini, or Copilot)
2. A consent modal will appear
3. Read the information about how VibeAI works
4. Check the checkbox to confirm understanding
5. Click "Start Reflecting" to enable VibeAI

### If You Click "Maybe Later"
- VibeAI will pause (not permanently disabled)
- A small "VibeAI is paused" bar appears
- Click "Enable" anytime to return to the consent modal
- Closing your browser resets this - you'll see the consent modal again

## Troubleshooting

### Extension Not Loading
- Make sure you selected the `dist` folder, not the parent folder
- Ensure Developer mode is enabled
- Try clicking the refresh icon on the extension card

### Consent Modal Not Appearing
- Hard refresh the page (Ctrl+Shift+R or Cmd+Shift+R)
- Check that you're on a supported platform
- Try disabling and re-enabling the extension

### HUD Not Showing
- Make sure you accepted consent (clicked "Start Reflecting")
- Wait 5-10 seconds after page load
- Try refreshing the page

### Extension Icon Shows Error
- Go to chrome://extensions/
- Click "Errors" on the VibeAI card
- Send the error details to hugonomysystems@gmail.com

## Uninstalling

1. Go to `chrome://extensions/`
2. Find "VibeAI FoldSpace"
3. Click "Remove"
4. Confirm removal

All local data is automatically deleted when you remove the extension.
