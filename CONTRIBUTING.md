Thank you for your interest in contributing to VibeAI FoldSpace!

## How to Report Issues

1. Check if the issue already exists in GitHub Issues
2. If not, create a new issue with:
   - Clear description of the problem
   - Steps to reproduce
   - Browser + version
   - AI platform (ChatGPT, Claude, etc.)

## Feature Requests

We welcome feature requests! Please open an issue with:
- Use case description
- How it would improve your workflow
- Any relevant examples or mockups

## Beta Testing

As a beta tester, your feedback is invaluable. Focus on:
1. Emotional analysis accuracy
2. Coaching usefulness
3. HUD UX (intuitive? intrusive?)
4. Performance impact
5. Real-world use cases
