# Changelog - VibeAI FoldSpace

All notable changes to this project will be documented in this file.

## [2.15.0] - 2026-02-01

### Added
- Multi-platform support: ChatGPT, Claude, Gemini, Copilot (Chrome)
- Live emotional tone analysis (Calm, Urgent, Reflect, Tension, Aligned)
- Real-time post-send coaching for urgent/confused prompts
- Conversation resonance tracking
- Resizable HUD (280-520px width, drag left edge)
- Privacy-first architecture (all processing local)
- HugoScore emotional analysis engine (obfuscated)

### Known Issues
- Edge browser blocks extensions on copilot.microsoft.com (Microsoft policy)
- Thread visualization limited on Copilot (single-conversation model vs multi-thread)
- WASM blocked on CSP platforms (JS fallback works identically)

### Workarounds
- Use Copilot in Chrome for full functionality
- Use ChatGPT, Claude, or Gemini in Edge (all work perfectly)

### Coming Soon
- March 2026: User-triggered injection for Edge/Copilot compatibility
- April 2026: Enhanced prompt library, FoldSpace overlay
- May 2026: Meta AI, Perplexity support
